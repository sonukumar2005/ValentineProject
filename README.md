# ❤️ Ask Her Out - Cute Web Project

## 📌 Description

This is a simple and fun web project designed to display a cute romantic message with an animated GIF.
It creates a playful and emotional interaction experience using HTML, CSS, and embedded media.

---

## ✨ Features

* 💖 Romantic message display
* 🐱 Cute animated Mochi cat GIF
* 🌐 Simple and lightweight web page
* 🎨 Styled using CSS

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Tenor GIF Embed

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
```

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. Enjoy the cute message 😄

---

## 📸 Preview

Displays a romantic message:

> "I knew it! You Love me a lot 😘"

With a cute Mochi cat GIF embedded from Tenor.
